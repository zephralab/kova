# Expenses API Tests

Test requests for expense-related endpoints (coming soon).

## Planned Endpoints

- `POST /api/projects/[id]/expenses` - Add expense
- `GET /api/projects/[id]/expenses` - List expenses
- `PATCH /api/expenses/[id]` - Update expense
- `DELETE /api/expenses/[id]` - Delete expense

## Structure

When implemented, this folder will contain:
- `requests.http` - REST Client format
- `add-expense.sh` - Add new expense
- `list-expenses.sh` - List project expenses
- `README.md` - Documentation
